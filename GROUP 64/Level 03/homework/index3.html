პროგრამირების ენები და მათი გამოყენება

Python

Python არის ყველაზე ცნობილი, მარტივი და მრავალმხრივი კოდირების ენა, ის შეიქმნა 1991 წელს გიუდო ვან როსუმის მიერ და იმის შემდეგ ვითარდება,იგი გამოიყენება back end-ში.

HTML

HTML-HyperText Markup language (ჰიპერტექსტური აღწერის ენა) , შეიქმნა ტიმ ბერნერს-ლის მიერ 1993 წელს, HTML გამოიყენება საიტების შესაქმნელად (რასაც ახლა მე ვაკეთებ) და საიტს აძლევს იმ სტრუქტურას რომელზეც შემდეგ შენდება CSS და JavaScrip.

CSS

CSS-Cascading Style Sheets (კასკადურ სტილთა ფურცლები) შეიქმნა წელს ჰოკონ ვიუმ ლის მიერ, ის გამოიყენება საიტების გასაფორმებლად და ზოგადად სტილის მისაცემად.

JavaScript

JavaScript არის პროგრამირების გამორჩეული ენა, საიტი რომ ბრძანებაზე რეაგირებს ეს JavaScript-ის დამსახურებაა, ის შეიქმნა 1995 წელს Netscape-ს მიერ.
